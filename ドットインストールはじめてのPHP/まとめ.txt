ブラウザはURLを入力するとインターネット上のコンピュータがそのファイルを
投げ返してくれる仕組み。
投げ返してくれる、つまりサーブしてくれるという意味でサーバーと呼ばれる。
PHPの拡張子の場合、PHPコードを実行してからHTMLが生成される。
ドッカーのメリットは手元のPCから開発環境を分離して管理できることと、他のPCで開発したものを動かしたい場合にドッカーを入れるだけで良いこと。

3
PHPのコードを埋め込むには  <?php  ?>  と記述する。
<?php echo 'こんにちは' . date('l'); ?>
echoは値を出力する
文字列は ''  もしくは ""で囲う
曜日を入れたい場合は文字列で小文字の l を入れる
命令文の最後に ; を入れる

4
変数の最初に $ を入れる

5
mt_rand(1,3) は乱数を生成する
この場合は1以上3以下

if (変数1 === ?) {
  変数2 = 'aaaaaa';
} else if (変数1 === ??) {
  変数2 = 'bbbbbb';
}
